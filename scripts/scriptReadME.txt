# Script Collection

This repository contains a set of Bash scripts for running various system and performance tests.

## Folder Structure

All scripts are located in the `scripts` directory.

## How to Run

To execute any script, make sure you are outside the `scripts` folder. Use the following command format:

```bash
./scripts/name_of_file.sh

Example
./scripts/compare_algorithms.sh

Output
Expected output for this would be a folder called output which will contain another folder called results which will contain results of experiments 


Script Descriptions

compare_algorithms.sh
Compares the performance of different scheduling algorithms.

context_switch_test.sh
Measures context switch performance and related metrics.

extract_metrics.sh
Extracts key performance metrics from system logs or output files.

quantum_test.sh
Tests and evaluates the impact of varying quantum values on scheduling performance.
